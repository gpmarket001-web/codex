# 🎬 PROTOCOLO DE VÍDEO UGC — Padrão Único (Linha Alenice × Higgsfield)

> **Módulo de vídeo do Cérebro da Linha Alenice.** Este é o ÚNICO modelo de produção de
> vídeo. Todo vídeo de todo produto passa por aqui — sem exceção. Funde o método da skill
> `higgsfield-content-factory` (absorvida) com nossos dados validados e o "aspecto campeão"
> descoberto em teste real.

**Versão:** 1.0 · **Público:** mulher 40+ (BR) · **Tráfego:** Google Ads (Meta inativo) ·
**Motor de escala:** Seedance 2.0

---

## 0. O princípio (o "SEMPRE")

**Um modelo só: UGC espontâneo fiel.** Nada de editorial polido de estúdio — em 2026, polimento
sinaliza "anúncio" e dispara ceticismo; autêntico converte. Pro 40+, o vídeo existe pra
responder UMA pergunta: *"será que cai bem em mim?"* — e só o **movimento** responde isso.

O aspecto campeão validado:

> **Foto real → movimento UGC espontâneo → valida grátis → recicla só o vencedor → CapCut → Google Ads.**

## 1. Herdado da skill `content-factory` (o método — inegociável)

| Regra | Como aplicamos |
|---|---|
| **UGC-first** | O formato `ugc` (pessoa + produto, momento de vida) é o eixo de tudo. |
| **ZERO texto no vídeo gerado** | Geração sai limpa, sem legenda/marca. Texto e hook entram **só no CapCut**. A "caption" é metadado de post, nunca overlay na geração. |
| **Identidade consistente obrigatória** | Mesma modelo entre clipes via **Seedance reference-driven** (a skill lista Seedance 2.0 como o motor de identidade consistente / multi-SKU). |
| **Research → Plano → Gate → Gera** | Nunca gerar no escuro. Pesquisa ancora, plano organiza, permissão antes de cada lote. |
| **Producibility ≤ 15s** | Todo clipe entre 4–15s. Narrativa maior = 2 cortes editados no CapCut. |
| **Negative prompt sempre** | Bloqueia texto/legenda/marca na geração (template na seção 7). |

## 3. As regras duras (valem em TODO vídeo)

1. **Fidelidade sagrada** — sempre da **foto real** do produto. Nunca text-to-image (reinventa a peça → troca/chargeback no 40+).
2. **Zero texto na geração** — hook/legenda/CTA entram no CapCut.
3. **Som OFF na geração** — locução PT-BR adicionada no CapCut (70%+ assiste sem som; o vídeo tem que vender mudo).
4. **Validar barato, reciclar caro** — valida em Kling Turbo (15 cr) ou no que já existe; análise é **grátis**; Seedance (≈22–35 cr) entra **só no campeão**.
5. **Identidade consistente** — mesma modelo entre clipes (Seedance reference).
6. **Hook nos 1,5s** — problema específico do 40+, não elogio vago. Ancorado nos nossos dados.
7. **Gate de permissão** — confirmar antes de cada lote pago.

---

## 4. O loop operacional (passo a passo — SEMPRE, todo produto)

```
1. FOTO REAL do produto (fornecedor / catálogo) ........... input que destrava tudo
2. GERA 3 ÂNGULOS UGC (caimento · giro · day-in-life) ...... Kling Turbo 720p 9:16 som off (15 cr/ea)
3. VALIDA GRÁTIS (virality_predictor) ..................... hook · retenção · viral  (custo ZERO)
4. CRAVA O CAMPEÃO ........................................ você lê os scores no painel
5. RECICLA SÓ O CAMPEÃO em Seedance 2.0 ................... identidade consistente, qualidade (1× gasto)
6. CAPCUT ................................................. hook 1,5s + locução PT-BR + CTA
7. GOOGLE ADS ............................................. campanha separada · 72h intocada · Proteção de Pixel
8. KAIZEN ................................................ o vencedor vira o molde do próximo produto
```

**Rate limit Plus:** 2 análises simultâneas. Enfileira — não trava o loop, só serializa.

## 6. Banco de narração (CapCut — hook → prova → CTA, som off)

## 7. Template de prompt (Seedance / Kling — colar e trocar o produto)

```
Animate the reference image into an authentic, natural-looking vertical UGC video.
An elegant Brazilian woman in her mid 40s wearing the [PRODUTO] from the reference image,
[MOVIMENTO: walks toward camera, pants swaying to show the drape | does a smooth 360 turn |
adjusts the piece and approves her look], relaxed candid home-video feel, soft natural
window light, no on-screen text.
Keep [PRODUTO] color, shape, waist and details faithful to the reference image.
Natural body movement, natural hands and face.
Negative: no text overlay, no captions, no subtitles, no on-screen typography, no watermarks,
no lower-third, no brand banners. Clean image only.
```

Config: `aspect_ratio 9:16` · `duration 10` · `generate_audio false` · `resolution 720p` (teste) / `1080p` (campeão).

## 8. O que fica OFF (honestidade — reativar quando fizer sentido)

- Formatos virais jovens da skill (challenge, street interview, ASMR) — **off** pro 40+.
- Marketing Studio presets + avatar preset — opcionais; nosso default é Seedance com foto real.
- Meta Ads scheduler (Stage 4 da skill) — **off** enquanto Meta estiver inativo.
- Cost-report (Stage 5) — rodar pontualmente pra mostrar economia vs produção tradicional.
